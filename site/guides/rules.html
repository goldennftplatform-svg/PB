Prize — pure SOL

Winners receive SOL from the jackpot treasury wallet (jackpot_tax). Game tokens are entry keys — the prize is lamports you can spend anywhere.

Each round's SOL commitment is recorded in the round ledger so the prize isn't confused with meme coin price action.

How you get in

No separate ticket purchase. Hold game tokens at snapshot time. Combined USD across all registered mints per wallet.

Trade tax → jackpot

Tax tokens harvest to the jackpot wallet, drip-sell to SOL in small chunks. LP fees can be claimed manually and forwarded. See liquidity guide.

Meme callout (rare)

Most rounds: SOL only. On special admin-enabled rounds, 10% of pre-snapshot SOL buys an announced meme mint before the draw outcome is known.

Provably fair

  • On-chain snapshot seed + Pepe count (1–30)
  • Hardened admin auth — only game-day admin can snapshot / settle
  • Holdings snapshot before draw
  • Every payout tx on Solscan
